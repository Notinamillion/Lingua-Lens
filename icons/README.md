# Extension Icons

This folder should contain three PNG icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

### Option 1: Use an Online Tool
1. Go to https://www.favicon-generator.org/
2. Upload or create a simple icon (book, language symbol, or "A→中" text)
3. Generate and download all sizes

### Option 2: Use Design Software
- Use Figma, Canva, or Photoshop
- Create a square canvas (128x128)
- Design your icon (suggested: book with translation symbol)
- Export as PNG in all three sizes

### Option 3: Simple Text Icon
Create a simple colored square with "A→中" or similar text representing language translation.

## Temporary Workaround

The extension will work without custom icons - Chrome will display a default placeholder icon until you add proper icons.

## Design Suggestions

**Color Scheme**: Purple/Blue gradient (matching the extension theme)
- Primary: #667eea
- Secondary: #764ba2

**Icon Ideas**:
- Book with language symbols
- Globe with translation arrows
- Letter "A" transforming to Chinese character
- Brain with multiple language symbols
